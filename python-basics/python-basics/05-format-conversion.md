# Format Conversion

### int, float, complex, boolean, string

* from int to float
```python
c = 10
print(float(c))
# 10.0
```

* from float to int
```python
a = 9.
print(int(a))
# 9
```

* from int to complex
```python
print(complex(3))
# (3+0j)
```

* from boolean to int
```python
print(int(True))
# 1

print(int(False))
# 0
```

* from string to int
```python
print(int('3'))
# 3
```

* from int to string
```python
print(str(77) + 'a')
# 77a
print(str(10.4) + 'a')
# 10.4a
```
