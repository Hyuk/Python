# Operations
```python
= + - / * ** //
```

## Classic Division ( / )
나눗셈 결과값 실수
```python
num1 = 7
num2 = 2

result = num1 / num2

result

# 3.5
```

## Multiplication ( * )
* 숫자 * 숫자
```python
num1 = 3
num2 = 2

result = num1 * num2

result

# 6
```

* 문자 * 숫자
```python
string1 = "Hello "
num1 = 5

result = string1 * num1

result

# 'Hello Hello Hello Hello Hello '
```

## Flost Division ( // )
나눗셈에서의 몫
```python
num1 = 7
num2 = 2

result = num1 // num2

result

# 3
```

## Remainder ( % )
나눗셈에서의 나머지
```python
num1 = 7
num2 = 2

result = num1 % num2

result

# 1
```
