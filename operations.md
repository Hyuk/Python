# Operations
```python
= + - / * ** //
```

## Classic Division ( / )
나눗셈 결과값 실수
```python
num1 = 7
num2 = 2

result = num1 / num2

result

# 3.5
```

## Flost Division ( // )
나눗셈에서의 몫
```python
num1 = 7
num2 = 2

result = num1 // num2

result

# 3
```

## Remainder ( % )
나눗셈에서의 나머지
```python
num1 = 7
num2 = 2

result = num1 % num2

result

# 1
```
